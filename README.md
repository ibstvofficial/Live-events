# 📺 Live event's - Auto IPTV Playlist

This repository automatically merges and updates sports and live TV playlists from multiple sources. It focuses on live events like SonyLiv, FanCode, and other sports channels, ensuring the links are always fresh.

## ✨ Features
* **Auto-Update:** Powered by GitHub Actions (updates every 6 hours).
* **Promotion Included:** Always features **IBS TV** official links at the top.
* **Fail-safe:** If a stream link is missing, it automatically redirects to a backup Telegram promo video.
* **Clean M3U:** Merged and organized for easy use in any IPTV player.
